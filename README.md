<p>Simple auth system using Mongo Express Nuxt Node.</p>
</div>

# install setup

```sh
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate
```
